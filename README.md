# HeadTailTransfer: An Efficient Sampling Method to Improve the Performance of Graph Neural Networks in Predicting Sparse ncRNA–Protein Interactions

This repository contains the official implementation of the paper:

> **HeadTailTransfer: An efficient sampling method to improve the performance of graph neural network method in predicting sparse ncRNA–protein interactions**
> **Jinhang Wei**, Linlin Zhuo, Shiyao Pan, Xinze Lian, Xiaojun Yao, Xiangzheng Fu.
> *Computers in Biology and Medicine*, 157 (2023) 106783.
> DOI: [10.1016/j.compbiomed.2023.106783](https://doi.org/10.1016/j.compbiomed.2023.106783)

> **Note on authorship:** HeadTailTransfer — the head-to-tail sampling strategy, implementation, experiments, and manuscript — was primarily developed and written by **Jinhang Wei** (co-first author).

This is the code for HeadTailTransfer.

# Requirements
Python 3.9
pytorch 1.13.0
pytorch-geometric 2.1.0
sklearn 1.1.3
torchmetrics 0.11.0

The main programs in the train folder are Head2Tail_main.py and HeadTransfer_main.py.
